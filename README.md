#1월 실행 로드맵: 구조적 설계 및 선형 시스템 기반 구축
1월의 최우선 목표는 파이썬의 객체 지향 문법을 체화하고, 바이오 데이터를 행렬 시스템(Ax=b)으로 치환하는 엔지니어링 감각을 확보하는 것입니다.

주간 세부 실행 과제
1주차 (1/1 ~ 1/7): 환경 구축 및 클래스 뼈대 설계
Math: MIT 18.06 L7-L8 (Ax=b의 해, Rank, Null Space).

Code: DNA_Sequence 클래스 초기 모델 구현.

init: 서열 입력 및 유효성 검사(ATGC 외 문자 필터링).

repr: 객체 출력 시 서열 정보 가독성 확보.

Memory: 단순 문자열 저장과 비트 연산을 이용한 저장 방식의 메모리 점유 차이 분석.

Output: 깃허브 레포지토리 생성 및 README.md 작성 후 첫 커밋 완료.

2주차 (1/8 ~ 1/14): 로직 모듈화 및 기능 확장
Math: MIT 18.06 L9-L10 (Independence, Basis, Dimension).

Code: 기능 확장 및 파일 분리.

get_reverse_complement: 상보적 역서열 생성 메서드 구현.

클래스 파일을 별도 모듈로 분리하고 main.py에서 import 하여 사용하는 구조 확립.

Output: 상보 서열 변환 기능 테스트 결과 및 모듈화된 코드 커밋.

3주차 (1/15 ~ 1/21): 행렬 연산의 본질 구현
Math: MIT 18.06 L11-L12 (Matrix Spaces, Graphs, Networks).

Code: Pure Python 행렬 연산 모듈 제작.

중첩 리스트와 리스트 컴프리헨션을 이용한 행렬 곱셈 함수 구현.

인접 행렬(Adjacency Matrix) 변환기 기초 설계.

Output: 두 행렬의 곱을 산출하고 결과의 차원을 검증하는 모듈 커밋.

4주차 (1/22 ~ 1/31): PS 완수 및 프로젝트 통합
Math: Problem Set 3, 4 풀이 및 오답 정리.

Code: 1월 통합 프로젝트 코드 완성 및 리팩토링.

Output: Bio-Matrix Mapping 시스템 최종 결과물 도출 및 README 업데이트.

1월 통합 프로젝트: Bio-Matrix Mapping & Network System
개요: 다수의 DNA 서열 간 유사도를 행렬로 수치화하여 데이터의 독립성을 분석함.

입력: FASTA 형식의 서열 리스트.

로직: 서열 간 Identity(일치도) 점수를 n x n 인접 행렬의 성분으로 배치.

결과: 생성된 행렬의 Rank를 계산하여 데이터 집합의 독립성(중복성) 여부를 리포트로 출력.

실전 제언 (Critical Level 60%)
폼 잡지 마십시오: 1월 1일 깃허브 레포지토리가 생성되지 않았다면 이 로드맵은 시작도 안 된 것입니다. 계획을 다듬는 시간에 코드 한 줄을 더 쓰십시오.

이론의 함정: L7 강의를 보고 이해했다고 착각하지 마십시오. 그 수식을 파이썬 if-else와 for-loop로 변환하지 못한다면 실제로는 아무것도 모르는 것입니다.
